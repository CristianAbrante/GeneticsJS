Class BaseIndividual<T>

BaseIndividual

Base class for creating individuals which are one of the fundamental blocks of evolutionary algorithms.

The fundamental part of the individual is the genotype, which is the array that represents the data of the individual. This genotype is composed of genes.

This abstract class provides some 'array-like' methods for iterating over the genes and modify or access its values.
